# Task Management System

# Overview

The Task Management System is a Java-based application built using Object-Oriented Programming (OOP) principles. It is designed to help users efficiently manage their daily tasks by providing functionalities such as adding, deleting, and updating tasks, along with priority management and notifications.

This project demonstrates practical implementation of core OOP concepts including Encapsulation, Inheritance, Polymorphism, and Abstraction.

---

 # Features

 ➕ Add Tasks

Users can create new tasks by providing relevant details such as title, description, due date, and priority.

 🗑️ Delete Tasks

Users can remove tasks that are no longer needed.

 🔄 Update Task Status

Tasks can be updated based on progress:

* Pending
* In Progress
* Completed

# ⚡ Priority Management

Each task can be assigned a priority level:

* High
* Medium
* Low

# 🔔 Notifications

The system provides notifications to help users stay updated about:

* Upcoming deadlines
* Overdue tasks
* Pending important tasks

---

# OOP Concepts Implemented

# Encapsulation

All task-related data is kept private within classes and accessed through methods, ensuring data security.

# Inheritance

Reusable code is achieved by creating hierarchical class relationships where child classes extend parent classes.

# Polymorphism

Methods are designed to perform different behaviors depending on the object, improving flexibility.

# Abstraction

Only essential functionalities are exposed to the user while internal implementation details are hidden.



# How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/task-management-system.git
```

2. Navigate to the project folder:

```bash
cd task-management-system
```

3. Compile the Java files:

```bash
javac *.java
```

4. Run the application:

```bash
java Main
```
